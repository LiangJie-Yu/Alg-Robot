## list列表
| 方法                               | 描述                                                                                |                                                                                                       |
| -------------------------------- | --------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| list.append(x)                   | 往列表里加**一整个东西**<br>（可以是列表，数字，字符串）                                                  | 想加一个元素                                                                                                |
| list.extend(L)                   | 把另一个列表**拆开**来一个个加进去                                                               | 想把另一个列表合并进来                                                                                           |
| list.insert(i, x)                | 在第i个元素**之前**插入一个x<br>a.insert(len(a), x) 相当于 a.append(x) 。                        |                                                                                                       |
| list.remove(x)                   | **删除**列表中**第一个**值为**x**的元素<br>如果没有这样的元素，就会直接报错，程序会停                               |                                                                                                       |
| list.pop([i])                    | 返回 **删除指定下标元素**的**值**<br>如没有指定索引，则返回最后一个元素<br>（ i 两边的方括号表示这个参数是可选的，而不是要求你输入一对方括号） |                                                                                                       |
| list.clear()                     | 移除列表中的所有项，等于del a[:]。                                                             |                                                                                                       |
| list.index(x, start, end)        | 返回列表中第一个值为 x 的元素的索引。如果没有匹配的元素就会返回一个错误。                                            |                                                                                                       |
| list.count(x)                    | 返回 x 在列表中出现的次数；不存在返回0                                                             | 返回值：非负整数（0 或正整数）                                                                                      |
| list.sort(key=排序规则, reverse=布尔值) | 对列表中的元素进行排序（直接修改原列表）。无返回值，输出None                                                  | `key`：可选，传入函数（如 `key=str.lower` 忽略大小写、`key=len` 按长度排序）；<br>`reverse`：可选，默认 `False`（升序），设为 `True` 则降序。 |
| list.reverse()                   | 列表原地反转，无返回值，输出None                                                                |                                                                                                       |
| list.copy()                      | 返回列表的浅复制（新列表地址不同但是内容地址相同）                                                         |                                                                                                       |
```python
a=[1,2]
a.append([3,4])
print(a)#[1,2,[3,4]]

a=[1,2]
a.extend([3,4])
print(a)#[1,2,3,4]

a=[1,2]
a.insert(1,5)
print(a)#[1,5,2]

a=[2,1,2]
a.remove(2)
print(a)#[1,2]

a=[2,1,2]
a.pop(0)
print(a)#[1,2]

lst = [10, 20, 30, 40] 
x = lst.pop() # 不写索引 
print(lst) # [10, 20, 30] 
print(x) # 40
lst = [10, 20, 30, 40] 
x = lst.pop(1) # 删除下标 1 
print(lst) # [10, 30, 40] 
print(x) # 20

lst1 = [1,2,3] 
lst2 = lst1 # 两个变量指向同一个列表（同一个盒子） 
lst1.clear() #地址为空
print(lst1) # []都为空
print(lst2) 

fruits = ["苹果", "香蕉", "苹果", "橙子"] 
idx3 = fruits.index("苹果", 1) # 从下标1开始找“苹果”（跳过第一个苹果） 
print(idx3) # 输出：2  
idx4 = fruits.index("苹果", 1, 3) # 在下标1~3之间找“苹果”（左闭右开，只查下标1、2）
print(idx4) # 输出：2  # 
fruits.index("橙子", 1, 2) # 抛出 ValueError: '橙子' is not in list

nums = [1, 2, 2, 3, 2, 4] print(nums.count(2)) # 输出：3

words = ["apple", "banana", "pear", "orange"] 
words.sort(key=len) # key=len 表示按字符串长度排序 
print(words) # 输出：['pear', 'apple', 'banana', 'orange']

nums = [1, 2, 3, 4, 5] nums.reverse() print(nums) # 输出：[5, 4, 3, 2, 1]（原列表被修改）

lst1 = [1, 2, 3]
lst2 = lst1.copy()
print("原列表lst1的地址：", id(lst1)) # 比如：140671221520128 
print("新列表lst2的地址：", id(lst2)) # 比如：140671221520256
print("lst1[0]的地址：", id(lst1[0])) # 比如：9789664 print("lst2[0]的地址：", id(lst2[0])) # 比如：9789664 
print("lst1[1]的地址：", id(lst1[1])) # 比如：9789700 print("lst2[1]的地址：", id(lst2[1])) # 比如：9789700
#列表地址不同，元素地址相同
```
注意：类似 insert, remove 或 sort 等修改列表的方法没有返回值
